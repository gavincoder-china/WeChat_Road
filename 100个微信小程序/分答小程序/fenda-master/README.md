目的是什么

* 自我学习，可以通过做一款微信小程序学习微信小程序相关的开发
* 和别人交流，光说不练假把式，欢迎批评指正
* 记录过程，方便其他人学习和建议

为什么是分答？

* 分答足够的简单－目前功能也多，不过主要功能没有那么复杂，不反感
* 分答已经有自己的webapp，获取资源及设计方便，直接上手
* 分答的功能基本上能够覆盖微信小程序的功能文档，练手和学习很合适
* 知道的人多,嘿嘿
