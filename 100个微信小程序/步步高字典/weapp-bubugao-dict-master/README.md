# 微信小程序－步步高字典

### 说明：

实现拍照查字功能，特色：
- 使用七牛云存储
- ES6语法

### 数据接口:

- https://api.getweapp.com/engine/qiniu/upload
- https://api.getweapp.com/engine/faceplusplus/recognizetext
- https://api.getweapp.com/engine/juhe/xhzd/query

### 目录结构：

- images — 存放项目图片文件
- pages — 存放项目页面文件
- utils — 存放项目数据获取，日期格式化等文件

### 开发环境：

微信web开发者工具 v0.11.112301
