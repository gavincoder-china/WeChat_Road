# 微信小程序--健康菜谱
基于微信小程序的一个查找检索菜谱的应用
### 功能
* 分类检索菜谱
* 发现最新菜谱
* 根据菜谱名搜索菜谱

### 用到的API
* 页面路由
* 列表渲染
* 带参数页面跳转
* navigator
* scrollview下拉加载更多
* ...
