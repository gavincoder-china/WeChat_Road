# 微信小程序－车源宝

### 说明：

实现车源宝客户端部分功能，包括首页，车源，寻车，登录等页面。

### 数据接口:

使用本地数据

### 目录结构：

- images — 存放项目图片文件
- pages — 存放项目页面相关文件，包括car_source,index,login,search_car,user等页面
- store — 存放数据接口文件
- utils — 存放时间格式化等文件

### 开发环境：

微信web开发者工具 v0.10.102800
