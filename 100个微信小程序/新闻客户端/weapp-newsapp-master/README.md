# 微信小程序 weapp-newsapp

## 介绍

利用微信小程序官方开发者工具开发，使用截获的搜狗搜索GET请求包有关微信平台热门新闻与关键词的数据生成的新闻列表浏览。

## 说明

* 首页到新闻列表详情页的路由跳转
* MODAL框绑定显示新闻详情

## 使用

* 克隆本项目代码到本地

```
git clone git@github.com:hijiangtao/weapp-newsapp.git
cd weapp-newsapp
```

* 打开微信开发者工具；
* 添加项目->选择本项目目录->编译执行；
