## 项目描述
- 基于微信开发的仿「ONE · 一个」的微信小程序
- 项目中的部分静态资源来自 [ReactNativeOne](https://github.com/lipeiwei-szu/ReactNativeOne) 项目

**需要开启开发环境不校验请求域名以及 TLS 版本**

## 功能
- 每日图文，往期列表，每月图文列表
- 每日文章（短篇，连载，问答），文章阅读，查看往期文章
- 音乐播放（部分音乐为虾米音乐，暂无法播放），音乐故事，歌词等
- 最近电影信息列表，查看电影故事，播放电影预告片
